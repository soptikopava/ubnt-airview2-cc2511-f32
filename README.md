## Hardware
AirView2 has:
- USB Type A
- MMCX for external antenna
- CC2511-F32 CPU
- green LED
- pin on PCB for Button
- pin PWR (power)
- RESET_N pin
- pin CLK
- pin DATA
- GND pin
